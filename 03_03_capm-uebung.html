III
Aufgabe 1: Beta-Faktor berechnen
βi = σiM / σM² = Kovarianz / Markt-Varianz

Berechnen Sie den Beta-Faktor für jedes Wertpapier:

Tipp

Der Beta-Faktor zeigt, wie stark ein Wertpapier auf Marktbewegungen reagiert.

  • β = 1:Bewegt sich wie der Markt
  • β > 1:Aggressiv (verstärkt Marktbewegungen)
  • 0 < β < 1:Defensiv (dämpft Marktbewegungen)
  • β < 0:Gegenläufig zum Markt
IV
Aufgabe 2: Erwartete Rendite laut CAPM
μCAPM = Rf + βi × (μM - Rf)

Berechnen Sie die risikoadäquate Rendite (Soll-Rendite) für jedes Wertpapier:

Tipp

Die CAPM-Formel sagt: "Für dieses Risiko (Beta) sollte der Markt diese Rendite zahlen."

Beispiel: Bei β = 0,5 und Marktrisikoprämie von 7%:

μCAPM = 3% + 0,5 × 7% = 3% + 3,5% =6,5%

V
Aufgabe 3: Jensen's Alpha berechnen
αi = μi,realisiert - μCAPM = Ist-Rendite - Soll-Rendite

Berechnen Sie Jensen's Alpha und interpretieren Sie das Ergebnis:

Tipp

Jensen's Alpha misst die Über- oder Unterrendite relativ zur SML:

  • α > 0:Outperformance Wertpapierunterbewertet
  • α < 0:Underperformance Wertpapierüberbewertet
  • α = 0:Fair bewertet (liegt auf der SML)
VI
Aufgabe 4: Treynor Ratio berechnen
Ti = (μi - Rf) / βi

Berechnen Sie die Treynor Ratio und vergleichen Sie mit dem Markt (TM =7,0):

Tipp

Die Treynor Ratio zeigt die Überrendite pro Einheit systematisches Risiko.

Der Markt hat per Definition: TM = (μM - Rf) / 1 = Marktrisikoprämie

  • T > TM:Besser als der Markt
  • T < TM:Schlechter als der Markt
